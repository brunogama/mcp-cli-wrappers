# CLI Wrappers - LLM Documentation

## Project Metadata
- **Name**: cli-wrappers
- **Type**: CLI Tool Collection
- **Language**: Python 3.10+
- **Package Manager**: uv (Astral) with PEP 723 inline script metadata
- **Purpose**: Token-efficient CLI wrappers for Model Context Protocol (MCP) servers

## Quick Reference
- **Total Tools**: 16 wrappers (7 standalone, 9 sub-projects)
- **Patterns**: 4-level progressive disclosure, PEP 723 scripts, MCP tool calling
- **CLI Framework**: Click (hand-crafted), argparse (generated)
- **HTTP Client**: httpx with 30s timeout
- **Dependencies**: httpx, pydantic, click, rich, fastmcp
- **Token Savings**: 60-70% vs monolithic help systems
