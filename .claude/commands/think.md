# /think - Sequential Thinking for Complex Problems

Use sequential-thinking-cli to break down complex problems into structured thought chains.

## Instructions

1. Take the user's question from $ARGUMENTS
2. Use the `sequential-thinking` skill in **Think** mode
3. Orchestrate a 3-5 step thought chain via `uv run sequential-thinking-cli/cli.py sequentialthinking`
4. Present synthesized answer with reasoning steps shown

If $ARGUMENTS is empty, ask the user what they want to think through.
