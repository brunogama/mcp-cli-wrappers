# CLI Wrappers - Quick Reference

**17 Python CLI wrappers for MCP servers**
**Location**: `~/cli-wrappers/`
**Token Efficiency**: 60-70% savings vs traditional MCP tool calling

---

## 4-Level Help (Every Tool)

```bash
uv run TOOL.py --help              # Level 1: Quick overview (~30 tokens)
uv run TOOL.py list                # Level 2: All functions (~150 tokens)
uv run TOOL.py info FUNCTION       # Level 2: Function details
uv run TOOL.py example FUNCTION    # Level 3: Working examples (~200 tokens)
uv run TOOL.py FUNCTION --help     # Level 4: Complete reference (~500 tokens)
```

## Common Commands

```bash
# Web
uv run crawl4ai.py scrape --url URL
uv run firecrawl.py scrape --url URL
uv run exa.py search "query"
uv run exa.py research "multi-step question requiring synthesis"
uv run exa.py code-search "implementation pattern or API example"
uv run ref-cli/cli.py search --query "docs"

# Code
uv run repomix.py pack_codebase PATH
uv run deepwiki.py ask --owner OWNER --repo REPO --question "?"
uv run semly.py outline PATH

# Dev
uv run xcode-cli/cli.py xcode_build --xcodeproj PATH --scheme SCHEME
uv run apple-docset-cli/cli.py search "NSView"
```
